# She Can Foundation

A modern and responsive website developed for **She Can Foundation**, an NGO dedicated to empowering women through education, mentorship, skill development, and community support.

## Overview

This project was created as part of a Frontend Development Internship Task. The website focuses on delivering an engaging user experience while highlighting the foundation's mission, programs, impact, and opportunities for volunteers to contribute.

## Features

* Responsive Design for Mobile, Tablet, and Desktop
* Dark Mode / Light Mode Toggle
* Smooth Scroll Animations
* Interactive User Interface
* Modern Hero Section
* About Us Section
* Programs & Initiatives Showcase
* Impact Statistics with Animated Counters
* Image Gallery
* Volunteer / Join Us Section
* Contact Form with Validation
* Back-to-Top Button
* Accessible and User-Friendly Design

## Technologies Used

* HTML5
* CSS3
* JavaScript
* TypeScript
* React
* Vite

## How to Run Locally

### Prerequisites

* Node.js
* npm

### Installation

```bash
git clone https://github.com/sathvika32427/She-Can-Foundation.git
cd She-Can-Foundation
npm install
```

### Start Development Server

```bash
npm run dev
```

Open the local URL displayed in the terminal (usually `http://localhost:5173`).

### Build for Production

```bash
npm run build
```
